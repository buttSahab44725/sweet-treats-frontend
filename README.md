# 🍰 Sweet Treats - Interactive Dessert Corner

A sweet, clean, and highly responsive frontend landing page for an online dessert shop. This project focuses on a smooth user experience, pastel aesthetic design, and dynamic interactive elements using pure vanilla technologies without external frameworks.

✨ **Live Demo:** [View Live Website](https://buttsahab44725.github.io/sweet-treats-frontend/)

---

## 🚀 Features

- **Interactive Cart Sidebar:** Click the Cart navigation item to smoothly slide out the shopping cart from the right side.
- **Dynamic Cart Counter & Calculations:** Add items to your cart to see real-time updates in the navbar count, detailed breakdown inside the sidebar, and automated price accumulation.
- **Item Modification:** Easily remove specific individual items directly from the cart layout using the delete interface.
- **Smooth Navigation:** Built-in seamless auto-scrolling transitions when navigating via the header links (Home, Menu).
- **8 Delicious Categories:** Includes modern treats (Cupcakes, Tarts, Cheesecakes) alongside traditional desserts (Rasmalai, Motichoor Laddoo) formatted cleanly via Flexbox layout structure.

---

## 🛠️ Tech Stack Used

- **HTML5:** Semantic markup structure.
- **CSS3:** Custom styling utilizing modern Flexbox alignments, transitions, and fixed position animations.
- **JavaScript (ES6):** Complete state tracking logic, DOM event listeners, and dynamic template literal generation for the UI layout.
